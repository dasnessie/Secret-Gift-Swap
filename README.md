# Secret Gift Swap

Match your Secret Santa without one person knowing the whole result, while also avoiding someone getting a gift for someone they recently matched with. Web application written in Flask.

Features include:

- Generate matching for any list of names
- Constraints avoid unwanted pairings
- Unique link to send to each participant, or overview link that works for everyone
- Participants have the option to see who they will be getting a gift from, if they feel better knowing that
